List each directory on this computer that contains your Radarr-managed movies. The app scans these folders when searching for existing downloads.

Use an override when Radarr reports a movie folder that this computer cannot access directly. Overrides translate Radarr's paths to the folders that are available locally.

  • Radarr path: copy the path exactly as it appears in Radarr, such as /mnt/Movies.
  • Local path: enter where that library lives on this machine, like /Volumes/Movies or ~/Movies.
  • Write one override per line using radarr/path => /local/path.

The app applies overrides first and then falls back to scanning the library paths listed above.
